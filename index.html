FDiag

A log-analysis toolkit to support diagnosis of failures from cluster message logs

View the Project on GitHub diag-toolkits/FDiag

Welcome to the FDiag log-analysis project page.

The FDiag project is an international collaboration between researchers from the University of Texas at Austin, the University of Warwick, the Texas Advanced Computing Centre, and Xyratex (now Seagate Technologies). FDiag is an open-source toolkit, written in C++, that extracts message templates from syslogs, rationalized message logs, IBM BlueGene logs and Cray-XT logs, and automatically generates reports that administrators of cluster systems (both large and small) can use to infer the likely causes of system failures.

The latest version of FDiag is version 3 and it will be release in May, 2015.

For further information, please get in touch with the following researchers: Edward Chuah, edwardchuah@acm.org, Arshad Jhumka, H.A.Jhumka@warwick.ac.uk, James C. Browne, browne@cs.utexas.edu, Bill Barth, bbarth@tacc.utexas.edu, Sai Narasimhamurthy, sai.narasimhamurthy@seagate.com .
